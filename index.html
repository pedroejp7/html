Parte 1: Pré-História da Web

Volar

Antes do surgimento do HTML, a World Wide Web era principalmente um ambiente acadêmico e de pesquisa. Tim Berners-Lee, um cientista da computação do CERN (Organização Europeia para Pesquisa Nuclear), desenvolveu a ideia de um sistema de hipertexto distribuído, que ele chamou de World Wide Web, em 1989.

Parte 2: O Conceito de Hipertexto

Voltar

Tim Berners-Lee e seu colega Robert Cailliau trabalharam em um sistema que permitia aos usuários navegar por documentos interligados através de links de hipertexto. Este conceito de hipertexto formou a base da Web como a conhecemos hoje.

Parte 3: O Surgimento do HTML

Voltar

Para tornar os documentos acessíveis através da World Wide Web, Berners-Lee desenvolveu uma linguagem de marcação simples chamada HTML (HyperText Markup Language). HTML permitia que os criadores de conteúdo estruturassem seus documentos com marcações para criar links, adicionar imagens e formatar texto.

Parte 4: Evolução do HTML

Voltar

Ao longo dos anos, o HTML evoluiu para incluir novos recursos e funcionalidades. As diferentes versões do HTML foram desenvolvidas e padronizadas pela W3C (World Wide Web Consortium), uma organização que Berners-Lee ajudou a fundar para supervisionar o desenvolvimento da Web.

Parte 5: Construindo um Site com HTML

Voltar

Para construir um site apenas com HTML, você precisa entender os princípios básicos dessa linguagem. Isso envolve aprender como usar elementos HTML para estruturar seu conteúdo, adicionar links, imagens e outros recursos.

Parte 6: Exemplo de Estrutura HTML

Voltar 
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Meu Primeiro Site</title>
            </head>
            <body>
                <header>
                    <h1>Bem-vindo ao Meu Site</h1>
                </header>
                <nav>
                    <ul>
                        <li><a href="#">Página Inicial</a></li>
                        <li><a href="#">Sobre</a></li>
                        <li><a href="#">Contato</a></li>
                    </ul>
                </nav>
                <main>
                    <section>
                        <h2>Sobre</h2>
                        <p>Esta é uma breve descrição sobre o meu site.</p>
                    </section>
                    <section>
                        <h2>Conteúdo</h2>
                        <p>Aqui está o conteúdo principal do meu site.</p>
                    </section>
                </main>
                <footer>
                    <p>&copy; 2024 Meu Site. Todos os direitos reservados.</p>
                </footer>
            </body>
            </html>

Parte 7: Praticando e Aprendendo

Voltar
